### 📋 快速導覽
- **生活規範**：想了解生活細則請點擊 **[生活公約](rules.md)**。
- **交通資訊**：想知道接駁車時間請點擊 **[交通與接駁](traffic.md)**。
- **外送訪客**：朋友來訪或取餐規則請點擊 **[外送與訪客規範](visitor.md)**。
- **財務試算**：計算月費與生活開支請點擊 **[💰 我的財務試算](https://aitchorgtw-eng.github.io/financial-calculator/)**。
- **緊急狀況**：遇到緊急事件請立即查看 **[🚨 緊急應變須知](emergency.md)**。

### 💡 如何快速使用？

1. **查看詳細目錄**：請點擊畫面 **「左下角」** 的 **三橫線圖示 (≡)**，就會跳出所有章節選單。
2. **快速搜尋內容**：點開選單後，最上方有一個 **「搜尋框」**。直接輸入您想找的字（例如：**車**、**菸**、**包裹**），系統會立刻列出相關規定。
3. **回到最上面**：如果您讀到一半想回頂部，直接點擊畫面最上方的 **「日日養生村」** 標題即可。
